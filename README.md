# E-Commerce Operations Dashboard + Smart Alerts

Automated e-commerce monitoring system that tracks orders, ad spend, and key metrics — with intelligent Slack alerts when KPIs cross danger thresholds.

## Status: In Progress

## Tech Stack
- **Automation:** Make.com
- **Data Store:** Google Sheets
- **Dashboard:** Looker Studio
- **Alerts:** Slack
- **AI Analysis:** OpenAI GPT-4o-mini
- **Simulated Sources:** WooCommerce webhooks, Meta/Google Ads data
