# Workflow 2: Ad Spend Tracker

## Overview
Reads daily ad campaign data and calculates CPA and ROAS for each campaign.

## Architecture
[Google Sheets: Search Ad Spend] → [Set Variables: CPA & ROAS] → [Google Sheets: Update Row]

## Metrics Calculated
- CPA (Cost Per Acquisition) = Spend / Conversions
- ROAS (Return on Ad Spend) = Revenue from Ads / Spend
